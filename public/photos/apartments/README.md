# Apartment Building Stock Photos

## How to Add Photos

1. Visit these free stock photo sites:
   - **Unsplash**: https://unsplash.com/s/photos/apartment-building
   - **Pexels**: https://pexels.com/search/apartment%20building/
   - **Pixabay**: https://pixabay.com/images/search/apartment%20building/

2. Download photos and save them to this folder with these naming conventions:
   - `apt-exterior-1.jpg` through `apt-exterior-10.jpg` (exteriors)
   - `apt-modern-1.jpg` through `apt-modern-10.jpg` (modern/luxury)
   - `apt-urban-1.jpg` through `apt-urban-10.jpg` (urban/city)
   - `apt-aerial-1.jpg` through `apt-aerial-5.jpg` (aerial/drone shots)

3. Recommended image specs:
   - Width: 1200-1920px
   - Format: JPG (optimized for web)
   - File size: Under 500KB each
